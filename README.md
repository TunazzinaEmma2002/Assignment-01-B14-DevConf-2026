DevConf 2026 Landing Page
This is the official landing page for DevConf 2026, a premier event designed to bring together engineers, founders, and tech enthusiasts. The project features a modern, responsive, and minimalist design.

🚀 Key Features
Modern Hero Section: Engaging typography and a clear call-to-action button.

Speaker Showcase: Beautifully designed profile cards for key conference speakers.

Pricing Section: Detailed breakdown of various ticket packages and benefits.

AI Hackathon: Dedicated section for the upcoming hackathon with event details.

Fully Responsive: Optimized for all devices (mobile, tablet, and desktop).

🛠 Tech Stack
HTML5

CSS3 (using Flexbox & Grid)

Font Awesome (for high-quality icons)

🎨 Design Philosophy
The landing page follows a minimalist and tech-forward aesthetic. It utilizes a professional blue-accent color scheme to create a clean and trustworthy experience for all users.
